> vimrc with my favorites plugins and settings


## 💻 pre-requisites

[vim](https://www.vim.org/),
[vim-plug](https://github.com/junegunn/vim-plug)

## 🚀 Installing

```bash
git clone https://github.com/meiazero/vimrc.git
```

## 🚀 Using

```bash
cd vimrc
```
```bash
cp .vimrc $HOME
```

## 📦 Plugins
```bash
vim
```
```bash
:PlugInstall
```
